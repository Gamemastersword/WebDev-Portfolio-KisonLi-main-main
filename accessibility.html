Accessibility Statement

This page explains the accessibility features I implemented in my website, and how they help users have a better and more inclusive experience.

Clear Color Contrast

I used a custom color scheme with strong contrast between text and background (e.g., dark text on a light mint background), which improves readability for all users, especially those with visual impairments.

Responsive Design

The layout is built using Flexbox, allowing the website to adapt to different screen sizes. This ensures accessibility and usability on both desktop and mobile devices.

Descriptive alt Text for Images

All images on the website include descriptive alt attributes. This helps screen readers describe the visuals to users with visual disabilities.

Accessible Navigation at the Top of the Page

The navigation bar is placed clearly at the top using semantic HTML, making it easy to locate and use. It also supports keyboard navigation, which benefits users who cannot use a mouse.
