1. Empathize

I considered the needs of different users, such as those using mobile devices. I researched accessibility guidelines and tried to understand how real users might interact with the site.

2. Define

I defined the main goal of the website: to showcase my work and personality in a way that is clear, engaging, and easy to navigate. I identified key accessibility and responsiveness challenges early on. Including the red color of the global goals, which is attractive.

3. Ideate

I brainstormed layout ideas, color themes, and content structures. I also considered interactive elements that would make the experience more dynamic and user-friendly.

4. Prototype

I created a first version of the website using HTML, CSS, and JavaScript. This included basic navigation, layout structure, and initial content for each page.

5. finish

After several steps, I finally finish my website.
