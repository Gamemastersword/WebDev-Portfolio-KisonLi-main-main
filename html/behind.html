My Process (Step-by-Step)

Step 1: Layout Design

I first planned out the homepage structure on paper, thinking about what sections I need and how to organize them.

Step 2: Initial Coding

I created the basic structure using HTML, CSS, and JavaScript to set up layout and navigation logic.

Step 3: Background and Content

I chose the background image and color scheme, then filled in my personal information and project summaries.

Step 4: Subpage Development

I wrote additional HTML files for each project, and connected them to the main page with proper links.

Step 5: Final Testing

I checked functionality on different devices and asked classmates to test usability and design clarity.
